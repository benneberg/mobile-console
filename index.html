DevConsole

A self-contained, mobile-first in-page debugger. Console interception, network monitoring, storage explorer, DOM inspector, REPL, and system info — all in one floating panel.

What's inside

  • Console — log/warn/error/info with timestamps, object pretty-print, REPL with history
  • Network — fetch + XHR interception, method badges, status, duration, headers detail
  • Storage — IndexedDB explorer (DB→Store→Records), LocalStorage & SessionStorage viewer
  • DOM — collapsible live tree, element picker with hover highlight, attribute inspector
  • System — screen, UA, performance memory, connection, battery, permissions
  • Settings — timestamps, auto-scroll, max entries, font size, theme, monitoring toggles
  • Drag-to-resize, error badge on FAB, auto-reinject on SPA navigation, REPL command history ↑↓
